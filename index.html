Tentang Produk

APA ITU GG POS
ENTERPRISE?

GG Pasoryan Enterprise POS adalah sistem kasir (Point of Sale) modern berbasis cloud yang dirancang khusus untuk bisnis kuliner, ritel, dan UMKM di Indonesia. Dibangun dengan teknologi terkini — React, Firebase Firestore real-time, dan arsitektur multi-platform — sistem ini memungkinkan Anda mengelola seluruh operasional bisnis dari satu layar, kapan saja dan di mana saja.

Tidak seperti sistem kasir konvensional yang terikat ke satu perangkat, GG POS berjalan secara bersamaan di browser, APK Android, dan EXE Windows — semua terhubung ke database yang sama secara real-time.

Tersedia dalam dua mode: produk siap pakai untuk UMKM umum, dan solusi custom enterprise untuk bisnis lebih kompleks seperti supermarket, restoran dengan sistem dapur, atau jaringan multi-outlet.

  • Browser di HP Android, tablet, dan desktop secara bersamaan
  • Aplikasi APK Android yang bisa diinstall tanpa Play Store
  • Aplikasi EXE Windows yang langsung jalan tanpa install driver
  • Semua perangkat terhubung ke database yang sama secara real-time
  • Data cloud Firebase — tidak hilang meski HP rusak atau hilang
  • Setup dalam 1×24 jam — tidak perlu IT atau server sendiri

12 MODUL
ENTERPRISE

Semua yang dibutuhkan bisnis Anda sudah tersedia. Tidak ada modul yang dibeli terpisah. Tidak ada fitur yang dikunci.

Kasir & Transaksi
Antarmuka POS cepat, pencarian produk, keranjang, diskon, PPN 10%, checkout multi-metode: Tunai, QRIS, Transfer, Kasbon.
📋
Riwayat & Retur
Lihat transaksi per hari/tanggal, edit transaksi, retur item dengan alasan, cetak ulang struk kapan saja.
⏸️
Pesanan Tertahan
Hold bill per meja atau pelanggan, lanjutkan kapan saja. Ideal untuk restoran, kafe, warung makan.
📒
Buku Piutang
Catat kasbon/tempo pelanggan, pantau saldo, terima cicilan, riwayat pelunasan lengkap.
📦
Inventaris & Stok
Master produk & SKU, kategori, harga jual & HPP, stok real-time, alert otomatis jika stok menipis (≤5 unit).
💸
Jurnal Pengeluaran
Catat semua kas keluar — operasional, gaji, bahan baku. Langsung terhubung ke laporan laba rugi.
📊
Laporan Eksekutif
Laba rugi real-time (hari/bulan/tahun/semua), KPI cards, produk terlaris, ekspor CSV, cetak A4.
📋
Buku Kartu Stok
Audit trail semua pergerakan stok masuk/keluar, HPP average cost otomatis, ekspor CSV.
👥
Multi Kasir & Role
Unlimited kasir, 3 role (Admin/Kasir/Owner), kontrol akses per fitur, tidak bisa manipulasi data.
🔄
Manajemen Shift
Buka/tutup shift dengan modal kas, kalkulasi selisih kas otomatis, laporan per shift.
🖨️
5 Metode Cetak Struk
Browser Print, Bluetooth BLE, RawBT Android, WiFi Network, Web Serial USB. Support semua thermal printer.
🎨

Solusi Custom Enterprise

KEBUTUHAN LEBIH KOMPLEKS?
KAMI BANGUN UNTUK ANDA

Demo yang tersedia adalah produk siap pakai untuk UMKM umum. Untuk bisnis yang membutuhkan sistem lebih canggih — kami kerjakan custom dari awal.

🏪
Supermarket & Minimarket
Sistem terintegrasi multi-departemen dengan kontrol gudang penuh
  • 👑
    Admin
    Kelola produk, harga, promo, kategori, user, dan konfigurasi sistem secara penuh.
  • 👁️
    Owner / Direktur
    Dashboard eksekutif — laba rugi, omzet per kasir, laporan gudang, pantau dari mana saja.
  • 🧾
    Kasir Penjualan
    POS checkout, scan barcode, multi metode bayar, cetak struk. Tidak bisa akses stok atau laporan.
  • 📦
    Gudang / Stok
    Terima & keluarkan barang, update stok masuk, cek stok per lokasi/rak, laporan pergerakan barang.
  • 📊
    Manager / Supervisor
    Approve retur, pantau performa kasir, laporan shift, akses terbatas sesuai divisi.
  • Scan barcode produk via kamera atau scanner USB
  • Sistem promo & diskon per kategori / waktu
  • Transfer stok antar gudang / outlet
  • Purchase order ke supplier
  • Laporan per outlet / cabang
🍽️
Restoran, Kafe & Cloud Kitchen
Dari kasir hingga dapur — pesanan langsung masuk layar chef secara real-time
  • 🧾
    Kasir / Waitress
    Input pesanan per meja, pilih item menu, tambah catatan khusus, kirim ke dapur otomatis.
  • 👨‍🍳
    Dapur (Kitchen Display System)
    Layar dapur menerima pesanan real-time. Chef tandai “sedang dimasak” dan “siap” — kasir/waitress dapat notifikasi langsung.
  • 🏷️
    Cetak Stiker Pesanan Otomatis
    Printer label/stiker kecil di dapur cetak otomatis per item pesanan untuk ditempel ke makanan atau minuman sebelum diantar ke meja.
  • 👑
    Admin / Owner
    Kelola menu, harga, kategori, laporan penjualan per meja/shift, pantau dari mana saja.
  • QR code per meja untuk self-order pelanggan
  • Multiple kitchen display (dapur basah & kering terpisah)
  • Stiker label pesanan otomatis per item (nama + meja + catatan)
  • Notifikasi real-time pesanan siap ke waitress
  • Menu digital dengan foto & deskripsi item
  • Laporan item terlaris per jam / shift

TEKNOLOGI TERBAIK

Frontend
React 18 + Vite + Tailwind CSS
Antarmuka cepat, responsif, dan modern di semua ukuran layar.
Database
Firebase Firestore (Google Cloud)
Real-time sync, aman, 99.9% uptime, backup otomatis oleh Google.
Auth
Firebase Anonymous Auth
Login instan tanpa email. Cukup pilih role dan masukkan password.
Hosting Web
Vercel Edge Network
Loading super cepat dari seluruh Indonesia berkat CDN edge global.
APK Android
PWA + TWA via PWABuilder
Ukuran APK kecil, performa native, tidak perlu Play Store untuk install.

Electron (Portable)
File tunggal portable — tidak perlu install, tidak perlu driver.
Print Engine
ESC/POS + Web Bluetooth API
Support semua merk thermal printer. 5 metode koneksi berbeda.
CI/CD
GitHub Actions + Vercel
Build EXE otomatis via GitHub Actions. Deploy web otomatis via Vercel.
